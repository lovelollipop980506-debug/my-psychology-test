# 🐶 킁킁 (KKng-KKng)

<img src="logo.png" alt="킁킁 로고" width="100" height="100" style="border-radius: 50%;">

> **당신의 취향을 '냄새로' 찾아주는 심리테스트 놀이터**

킁킁은 강아지가 냄새를 맡듯이, 당신의 숨겨진 성격·취향·패턴을 재밌게 찾아주는 밈 기반 취향 심리테스트 플랫폼입니다.

## 💡 킁킁의 핵심 특징

### 🐾 귀엽고 직관적인 UI
- 따뜻한 파스텔톤 + 강아지 '킁킁' 캐릭터
- 누구나 보기 쉽고, 한 번 보면 기억 남는 브랜드 느낌
- 모바일 최적화로 언제 어디서든 빠르게 테스트 가능

### ⚡ 3분 안에 끝나는 초간단 심리테스트
- 10문항 내외의 짧고 정확한 테스트
- 고민 없이 클릭만 하면 결과 도출
- 부담 없이 바로바로 다른 테스트도 이어서 즐길 수 있음

### 😂 밈 기반 결과로 '공유 욕구 폭발'
- 친근하고 웃긴 결과 문구
- 친구들이랑 비교하면 더 재밌는 타입 분석
- 결과 페이지에 바로 '공유' 기능 탑재
- 즉시 공유해서 단톡방 난리남

### 🍱 다양한 테스트 카테고리
- Food MBTI, 연애 성향, 성격 유형 등
- 음식, 연애, 취향, 성격, 라이프스타일
- 시즌/밈 기반 테스트도 계속 업데이트
- "나 닮은 음식은?", "연애 세포 레벨?" 등 친구들이 좋아하는 포맷 중심

### 🚀 누구나 쉽게 즐길 수 있는 '바로가기 사이트'
- 회원가입 없이 즉시 시작
- 클릭 한 번으로 테스트 → 공유까지
- 심심할 때마다 와서 놀 수 있는 "가벼운 플레이그라운드"

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Pretendard 폰트, 반응형 디자인, 애니메이션 효과
- **Deployment**: GitHub Pages (정적 호스팅)

## ✨ 주요 기능

### 메인 페이지
- 테스트 목록을 카드 형식으로 표시
- 깔끔하고 직관적인 UI
- 킁킁 로고와 브랜딩 적용

### Food MBTI 테스트
- ✅ **커스텀 이름 입력 화면**: prompt 대신 디자인된 화면으로 이름 입력
- ✅ **개인화된 결과**: "{이름}님과 닮은 음식은?" 형식으로 결과 표시
- ✅ **URL 기반 결과 공유**: 친구에게 공유 시 개인화된 링크 생성
- ✅ **공유받은 결과 화면**: 공유받은 링크로 접속 시 다른 UI 표시
- 10개의 질문으로 구성
- 4가지 음식 타입 결과 (매운 라면, 초코 케이크, 군만두, 아이스 아메리카노)
- 실제 음식 사진과 함께 결과 표시

## 🚀 로컬 실행 방법

1. 이 저장소를 클론합니다:
```bash
git clone <repository-url>
```

2. `index.html` 파일을 브라우저로 엽니다:
   - 파일 탐색기에서 `index.html`을 더블클릭
   - 또는 Live Server 등의 로컬 서버 사용

## 📦 배포

GitHub Pages를 통해 배포할 수 있습니다:
1. GitHub 저장소 생성
2. 파일 업로드
3. Settings > Pages에서 "Deploy from a branch" 선택
4. main 브랜치의 루트(/) 선택
5. Save 클릭
